The idea of this talk is to present and discuss scientific applications for data analysis developed using C++ and Qt. A set of toolkit structures will be presented that are aimed at the visualization, analysis and manipulation of different types of data, focusing mainly on datasets related to big data. In addition, some details regarding the implementation of marK will be demonstrated: a tool for data manipulation and construction of datasets for machine learning that has been developed by the KDE Community.
